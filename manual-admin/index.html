管理者（システム管理者）向け操作マニュアル

このページは、Traingo のシステム管理者が全社の企業・コース・受講者・動画・アフィリエイト・研修企画書生成を操作するためのマニュアルです。

管理者は全企業を横断して管理できる最上位権限を持ちます。企業アカウントの作成・削除・slug設定・決済許可フラグなど、会社管理者にはできない操作が含まれています。

推奨の順番は ① 企業管理で企業追加 → ② コース管理でコース作成 → ③ 企業詳細で受講者追加 → ④ ダッシュボードで全社進捗確認 です。

上の検索ボックスに言葉を入れると、関係する項目だけに絞り込めます。画面右はしの縦書き「フィードバック」ボタンから不具合・ご意見をいつでも送れます。

この画面でできること

  • 全社サマリー：総企業数・総受講者数・受講中の人数・修了者数を一目で確認できます。
  • 企業別集計：契約中の各企業ごとに受講者数・修了者数・修了率を確認できます。
  • 進捗グラフ：企業別の受講状況が棒グラフ・数値で表示されます。

操作手順

  1. 左メニューの ⑤ 受講状況ダッシュボード を押します。
  2. 画面に全社サマリー（総企業数・総受講者数・受講中・修了）が表示されます。
  3. 企業別の集計表で、各企業の受講者数・修了者数を確認します。
  4. 特定企業の詳細を見たい場合は「① 企業管理」から企業を選び、企業詳細画面で受講者一覧を確認します。
注意この画面は確認専用です。数字の編集・受講者の操作はできません。企業ごとの詳細管理は「企業詳細」画面で行います。

この画面でできること

  • 企業一覧を確認：登録済みの全企業が一覧で表示されます。企業名・slug（URLに使う識別子）が確認できます。
  • 新規企業アカウントを作成：「+ 新規企業を追加」ボタンから新しい企業を登録できます（管理者専用の操作）。
  • 企業詳細へ移動：「詳細」ボタンから各企業の詳細画面に移り、担当者情報・受講者・決済許可設定ができます。

新規企業を追加する手順

  1. 画面右上の + 新規企業を追加 を押します。図の 1
  2. 企業名・担当者メールアドレス・初期パスワードを入力します。
  3. 企業を作成 を押すと、企業アカウントが作成され一覧に追加されます。
  4. 企業担当者にログインURLとメールアドレス・パスワードを伝えます。
入力欄必須/任意内容
企業名必須企業・組織の正式名称
担当者メールアドレス必須企業管理者のログインID（メールアドレス）
初期パスワード必須最初にログインするためのパスワード。8文字以上推奨
slug任意URLに使う識別子（半角英小文字・数字・ハイフン）。空欄は自動生成
管理者専用操作企業アカウントの作成は管理者のみが行えます。会社管理者ユーザーは自分の企業しか操作できません。企業を削除する場合は、先に受講者データのエクスポート・バックアップを行うことを推奨します。

この画面でできること

  • 企業情報の編集：企業名・担当者名・担当者メールアドレスを変更できます。
  • 決済リンク許可フラグの切替：この企業が受講者向けに決済リンクを使えるかどうかを管理者が制御します（管理者専用の権限）。
  • slug の設定・変更：URLに使う識別子を設定します（半角英小文字・数字・ハイフンのみ）。
  • 受講者一覧の確認：この企業に属する受講者の一覧が表示されます。各受講者の進捗・コース確認も可能です。
  • 受講者の追加：この企業に受講者を直接追加できます（メールアドレス・パスワード・担当コースを指定）。

企業情報を編集する手順

  1. 「企業管理」一覧で対象企業の 詳細 を押します。
  2. 企業名・担当者名・担当者メールアドレス・slugを必要に応じて書き換えます。
  3. 決済リンク許可の切替が必要な場合は、該当ボタンを押して状態を変更します。
  4. 保存 を押して変更を確定します。

受講者を追加する手順

  1. 企業詳細画面の「受講者一覧」セクションにある + 受講者を追加 を押します。
  2. 氏名・メールアドレス・初期パスワード・担当コースを入力します。
  3. 追加する を押すと受講者アカウントが作成されます。
  4. 受講者にログインURL・メールアドレス・パスワードを伝えます。
項目説明
決済リンク許可フラグ管理者のみ変更可。ONにするとこの企業の管理者が受講者向け決済リンクを発行できます
slugURLに使う識別子。a-sha と設定すると /company/a-sha/... のURLで企業管理画面にアクセスできます。変更時は既存のリンクが無効になるため注意
slug 変更時の注意slug を変更すると、その企業の管理画面URLおよび受講者のアクセスURLがすべて変わります。変更後は企業担当者に新しいURLを必ず伝えてください。

4コース管理（コース一覧・新規作成・公開設定）

全コースの一覧確認、新規コースの作成、公開/非公開の切替、コース削除を行う画面です。

人材開発支援研修コース公開中 詳細 削除
IT基礎研修非公開 詳細 削除

この画面でできること

  • コース一覧の確認：全コースとその公開状態（公開中／非公開）が一覧で表示されます。
  • 新規コースの作成：「+ 新規コース作成」からコース名・説明を入力して新しいコースを追加します。
  • 公開/非公開の切替：各コースの「公開中」「非公開」ボタンを押すと状態が切り替わります。非公開のコースは受講者画面に表示されません。
  • コースの削除：「削除」ボタンで該当コースを削除できます（確認ダイアログが出ます）。受講者がいるコースを削除するとデータが失われるため、事前に確認が必要です。
  • コース詳細へ移動：「詳細」ボタンからコース内容（章・レッスン）の編集画面に移動します。

新規コースを作成する手順

  1. 画面右上の + 新規コース作成 を押します。図の 1
  2. コース名（必須）と説明（任意）を入力します。
  3. 公開状態を「公開」または「非公開」から選びます（後から変更可能です）。
  4. コースを作成 を押すと作成完了です。
  5. 作成後はコース詳細（画面5）から章・レッスンを追加します。

コースを削除する手順

  1. 削除したいコースの 削除 を押します。
  2. 「このコースを削除しますか？」という確認ダイアログが表示されます。
  3. 内容を確認して 削除する を押すと削除されます。
削除前の確認コースを削除すると、そのコースに登録されていた受講者のデータ・進捗も削除されます。削除前に必要なデータをCSVなどでエクスポートしてください。削除は元に戻せません。

この画面でできること

  • 章（セクション）の追加・編集・削除：コースを章単位で構成します。章名・公開日時を設定できます。
  • レッスンの追加：章の中に動画・文章・クイズの3種類のレッスンを追加できます。
  • レッスンの並び替え：ドラッグ操作で章内のレッスン順序を変更できます。
  • レッスンの編集・削除：各レッスンの内容・タイトルを編集、または削除できます。

章（セクション）を追加する手順

  1. 画面下部の + 新しい章を追加 を押します。
  2. 章の名前（例：「第1章 研修の基礎」）を入力します。
  3. 公開日時を設定する場合は日時を入力します（空欄にすると即時公開）。
  4. 章を作成 を押します。

レッスンを追加する手順

  1. 追加したい章の + レッスン追加 を押します。
  2. レッスンの種類を「動画」「文章」「クイズ」から選びます。
  3. レッスン名を入力し、種類に応じてコンテンツを設定します。
    動画：動画管理に登録済みの動画を選択するか、YouTubeのURLを入力します。
    文章：エディタで本文を入力します。
    クイズ：問題文・選択肢・正解・合格点を設定します。
  4. レッスンを保存 を押します。
章・レッスンの削除について削除したレッスンの受講者の進捗データも失われます。受講中の受講者がいる場合は削除前に確認してください。

  • 進捗の詳細確認：各レッスンの完了状況（済／未）を一覧で確認できます。
  • 修了日の確認：コース修了日（全レッスン完了日）が表示されます。

  3. 複数コース受講者の場合はコースごとのタブを切り替えて確認します。
  4. 各レッスンの行に「視聴時間」「完了状況」が表示されています。
助成金証跡として使う場合人材開発支援助成金の証跡として視聴ログを提出する場合は、画面を印刷またはスクリーンショットで保存してください。

動画をアップロードする手順

?困ったとき（FAQ・お問い合わせ）

よくある質問と対処方法をまとめています。解決しない場合は画面右のフィードバックボタンからご連絡ください。

よくある質問（FAQ）

管理者ログイン画面のURLがわからない
管理者ログインのURLは /admin-login です。一般受講者のログイン（/login）とは別ページです。URLを忘れた場合はシステム担当者に確認してください。
企業を追加したが、企業担当者がログインできないと言っている
企業追加時に設定したメールアドレスとパスワードが正しく伝わっているか確認してください。ログインURLは /login（受講者・企業管理者共通）です。パスワードを忘れた場合は「企業詳細」画面から担当者のパスワードを再設定できます。
コースを公開にしたのに受講者画面に表示されない
コースの公開状態が「公開中」になっているか再確認してください。また受講者がそのコースに紐づけられているか「受講者管理」で確認してください。コースと受講者が紐づいていないと受講者画面には表示されません。
動画をアップロードしたがレッスンで選択できない
アップロード完了後、コース詳細画面でレッスン追加を行う際に「動画を選択」からアップロード済み動画を選べます。アップロード直後は一覧に反映されるまで数秒かかることがあります。ページを再読み込みして再度確認してください。
受講者の視聴時間が 0 のまま記録されない
受講者が動画を再生した際にネットワーク接続が不安定だった可能性があります。受講者に「再生中はネットワーク接続を切らないようにする」よう案内してください。また動画の再生が一定時間（通常30秒以上）続かないとログが記録されない場合があります。
slug を変更したら企業管理者から「ページが見つからない」と連絡があった
slug を変更すると企業管理者のアクセスURLが変わります。変更後の新しいURLを企業担当者に必ず伝えてください。ブラウザのブックマークも更新が必要です。
削除したコースのデータを元に戻したい
削除したコース・レッスン・受講データは元に戻せません。重要なデータは削除前に必ずエクスポート・スクリーンショットなどで保存しておいてください。
研修企画書生成の「企画書を生成する」を押してもエラーになる
AI の APIキーが設定されていないか、API利用制限に達している可能性があります。システム担当者に連絡してAPIの状態を確認してもらってください。
