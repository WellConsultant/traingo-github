Traingo 受講者 操作マニュアル

このマニュアルは、Traingo（研修システム）を使って研修を受講するための操作案内です。ログインからコース修了・修了証の取得まで、すべての操作手順を省略なく掲載しています。

はじめての方は ★ 画面の全体像 から読み進めてください。特定の操作を調べたい場合は左のメニューを押すか、上の検索ボックスにキーワードを入力してください。

💡 各セクションの「画面イメージ」はブラウザの表示を模した図です。自分の画面と見比べながら操作してください。右はしの フィードバック ボタンから、マニュアルへのご意見をいつでも送れます。

ログインすると、上部にメニューバー、中央に各画面のコンテンツが表示されます。どの画面にも共通のナビゲーションメニューから各ページへ移動できます。

1ダッシュボード

ログイン後に最初に表示される画面です。受講中のすべてのコースの進捗が一目で確認できます。

この画面でできること

  • 受講中コースの一覧表示：割り当てられているすべてのコースがカード形式で並ぶ
  • 進捗率の確認：各コースカードにパーセント表示と進捗バーが出る
  • 続きから受講：コースカードを押すと前回の続きから受講できる
  • 全体統計の確認：受講中コース数・完了レッスン数・修了済みコース数を一目で把握できる

操作手順（コースを開いて受講を続ける）

  1. ログイン後、自動的にダッシュボード画面が開く
  2. 受講したいコースのカードを見つける
  3. カードをクリック（またはタップ）するとコース詳細画面が開く
  4. コース詳細画面から受講を続けるレッスンまたは動画を選んで開始する
進捗バーについて 進捗バーはコース内の全レッスン・動画・クイズのうち、完了した数の割合を示します。すべて完了すると 100% になり、修了証が発行されます。

2コース・動画を見る

コース一覧でコースを選び、コース詳細画面から各章・動画を開いて視聴します。動画を最後まで見ると進捗が自動的に記録されます。

コース一覧画面（/course）でできること

  • 受講可能なコースを一覧表示：割り当てられたすべてのコースが並ぶ
  • コースの概要確認：コース名・説明・レッスン数・受講者数・進捗バーが表示される
  • コースを選んで詳細を開く：クリックするとコース詳細ページに進む

コース詳細画面（/course/[コースID]）でできること

  • コースの説明を読む：コースの目的・概要・学習内容が書かれている
  • 章・レッスンの構成を確認：どの章にどのレッスン・動画があるかを一覧で見る
  • 各レッスン・動画を選んで開く：完了済みのものはチェックマークで分かる

動画視聴画面（/video/[動画ID]）でできること

  • 動画を再生・一時停止：動画プレイヤーを操作して視聴する
  • 進捗の自動記録：動画を最後まで視聴すると自動的に「完了」として記録される
  • コメント・質問の投稿：動画ページ下部のコメント欄から管理者に質問できる
  • 次のレッスンへ移動：完了後に次のレッスン・動画へ進むボタンが表示される

操作手順（コース開始から動画視聴まで）

  1. 上部メニューの コース一覧 をクリックする
  2. 受講したいコースのカードをクリックしてコース詳細画面を開く
  3. 受講する章の中から見たい動画またはレッスンをクリックする
  4. 動画プレイヤーが開いたら再生ボタン（▶）を押して視聴を開始する
  5. 動画を最後まで視聴すると、自動的に「完了」として進捗が記録される
  6. 画面下部の 次のレッスンへ をクリックして次に進む
スキップ・早送りについて 助成金要件により、一部のコースは動画のスキップや途中飛ばしが制限されています。動画は最初から最後まで視聴してください。スキップできない場合は仕様のため、焦らずそのままご覧ください。
途中で画面を閉じた場合 動画を途中で止めてページを閉じても、次回ログイン時に続きから再生できます。進捗は定期的に自動保存されています。
YouTube / Vimeo の動画の場合 一部のコースでは YouTube または Vimeo の動画が埋め込まれています。視聴後に 完了にする ボタンを手動で押して完了を記録してください（自動記録されません）。

3レッスン（文章・クイズ）

コースには動画以外に、文章を読むレッスンや問題に答えるクイズが含まれることがあります。それぞれ完了の記録方法が異なります。

文章レッスン画面（/lesson/[レッスンID]）

  • テキストと画像で構成された学習コンテンツを読む
  • 読み終えたら 読み終えた ボタンを押す（ページ末尾にある）
  • ボタンを押した時点で「完了」として進捗に記録される

文章レッスンの操作手順

  1. コース詳細画面からレッスンをクリックして開く
  2. ページに表示されているテキストと画像を上から順に読む
  3. 一番下までスクロールして 読み終えた ボタンを押す
  4. 進捗バーが更新されていることを確認する
注意 読み終えた ボタンを押さないと完了として記録されません。スクロールするだけでは記録されないので、必ずボタンを押してください。

クイズ画面（/lesson/[レッスンID]）

  • 選択肢形式または記述形式の問題に答える
  • 回答後、その場で正解・不正解と解説が表示される
  • 全問回答して 提出する ボタンを押すと完了が記録される
  • 合格・不合格に関係なく、提出すると「完了」として記録される
  • 何度でも再挑戦できる（再挑戦しても進捗記録は上書き更新される）

クイズの操作手順

  1. コース詳細画面からクイズをクリックして開く
  2. 問題文を読み、選択肢をクリックして回答する
  3. すべての問題に回答したら 提出する ボタンを押す
  4. 合否と正解・解説が画面に表示される
  5. 再挑戦したい場合は もう一度挑戦する ボタンを押す

4設定（プロフィール・購入履歴・管理者への連絡）

上部メニューの「設定」ページでは、自分の登録情報の変更・購入履歴の確認・管理者への連絡ができます。

この画面でできること

  • プロフィール編集：氏名・メールアドレスを変更する
  • パスワード変更：新しいパスワードを設定する
  • 購入履歴の確認：過去にコースを購入した場合の履歴を確認する
  • スクール管理者への連絡：担当者にメッセージを送る

プロフィール（氏名・メールアドレス）を変更する手順

  1. 上部メニューの 設定 をクリックする
  2. 「プロフィール情報」欄の氏名またはメールアドレスの入力欄をクリックして新しい内容を入力する
  3. 保存する ボタンをクリックする
  4. 「保存しました」と表示されたら完了

パスワードを変更する手順

  1. 設定ページを開く
  2. 「パスワード変更」欄に現在のパスワードを入力する
  3. 新しいパスワードを「新しいパスワード」欄と「確認用」欄に同じ内容で入力する
  4. パスワードを変更する ボタンをクリックする
  5. 「パスワードを変更しました」と表示されたら完了
パスワードの条件 パスワードは8文字以上で設定してください。他のサービスと同じパスワードの使い回しは避け、英字・数字を組み合わせたものにすることをお勧めします。

購入履歴を確認する手順

  1. 設定ページを開く
  2. 「購入履歴」タブ（または該当セクション）をクリックする
  3. 購入したコース名・日付・金額の一覧が表示される
会社経由で受講している場合 会社（スクール）の管理者が費用を負担している場合、購入履歴は表示されないことがあります。領収書等が必要な場合は管理者にお問い合わせください。

スクール管理者（担当者）に連絡する手順

  1. 設定ページを開く
  2. 「管理者への連絡」欄にメッセージを入力する
  3. 送信する ボタンをクリックする
  4. 「送信しました」と表示されたら管理者に届いた状態になる
返信について 管理者からの返信は管理者側の画面で行われます。返信があった場合は、次回ログイン時に連絡内容のやり取りを確認してください。

5修了証

コース内のすべてのレッスン・動画・クイズを完了すると修了証が発行されます。修了証一覧ページで確認・印刷・保存ができます。

この画面でできること

  • 修了証一覧の確認：修了したすべてのコースの修了証が並ぶ（/certificates）
  • 個別の修了証を開く：コース名をクリックすると個別の修了証ページが開く（/certificate）
  • 修了証の印刷・保存：ブラウザの印刷機能でPDFとして保存・印刷ができる

lms-hojokin.vercel.app/certificates

修了証を確認・印刷する手順

  1. 上部メニューの 修了証 をクリックして一覧ページを開く
  2. 印刷・確認したい修了証のコース名をクリックする（または 修了証を開く ボタンを押す）
  3. 個別の修了証ページが開き、氏名・コース名・修了日が表示される
  4. 印刷したい場合はブラウザのメニューから「印刷」を選ぶ（Windowsは Ctrl + P、Macは Command + P）
  5. 印刷ダイアログで「PDFに保存」を選ぶとPDFファイルとして保存できる
修了証が一覧に出てこない場合
  • コース内にまだ完了していないレッスン・動画・クイズが残っている可能性があります
  • コース詳細画面を開いて、すべての項目にチェックマークがついているか確認してください
  • 動画を最後まで見たあとに「完了」ボタンを押し忘れた場合は、その動画を再度開いてボタンを押してください
助成金申請への使用について 修了証には氏名・コース名・修了日が記載されています。人材開発支援助成金（事業展開等リスキリング支援コース）の申請書類として提出する場合は、担当者の指示に従って提出してください。

？よくある質問

操作中に困ったときは、まず以下の質問と回答を確認してください。それでも解決しない場合は「設定」ページから管理者に連絡してください。

ログインできません。パスワードを忘れました。

ログインページの「パスワードをお忘れの方」リンクをクリックしてください。登録しているメールアドレスを入力すると、パスワード再設定のメールが届きます。メールが届かない場合は、迷惑メールフォルダも確認してください。それでも届かない場合は、会社の管理者（担当者）に連絡してパスワードのリセットを依頼してください。

動画が再生されません。

まずページを再読み込み（Windowsは F5、Macは Command + R）してください。それでも再生されない場合は、別のブラウザ（Chrome、Edge、Safariなど）で試してください。ネットワーク接続が不安定な場合も再生が止まることがあります。Wi-Fiや有線接続の状態を確認してください。

動画を最後まで見たのに「完了」として記録されません。

YouTube / Vimeo 埋め込み動画の場合は、視聴後に 完了にする ボタンを手動で押す必要があります。ボタンがない場合は一度ページを再読み込みして、動画を最後まで再生し直してみてください。

進捗率が更新されません。

ページを再読み込みすると最新の状態が反映されます。数分待っても変わらない場合は、コース詳細画面で未完了のレッスン・動画がないかを確認してください。

クイズで不合格になりました。もう一度受けることはできますか？

はい、何度でも再挑戦できます。クイズ結果画面の もう一度挑戦する ボタンを押してください。再挑戦後の結果で進捗が更新されます。不合格でも「提出」した時点で完了として記録されます。

修了証が発行されません。

コース内のすべてのレッスン・動画・クイズを完了しているか確認してください。コース詳細画面を開いて、すべての項目にチェックマークがついていれば修了証が発行されているはずです。それでも表示されない場合は、ページを再読み込みしてから修了証ページ（/certificates）を開いてください。

自分の名前やメールアドレスが違います。変更できますか？

はい、設定ページ（/settings）のプロフィール欄から変更できます。変更後は 保存する ボタンを押してください。メールアドレスを変更すると、次回ログイン時は新しいメールアドレスでログインすることになります。

動画を途中で閉じてしまいましたが、続きから見られますか？

はい、次回ログイン後に同じ動画を開くと続きから再生できます。進捗は定期的に自動保存されています。

スマートフォンやタブレットでも使えますか？

はい、スマートフォン・タブレットのブラウザからも受講できます。ただし、動画再生は安定したWi-Fi環境での利用を推奨します。

上記で解決しない場合 「設定」ページの「管理者への連絡」欄から、状況を詳しく書いてメッセージを送ってください。可能であれば、エラーが出ている場合は画面のスクリーンショットも添付すると、より早く対応できます。
